Black Myth: Wukong

Black Myth: Wukong é um jogo de RPG de ação que está sendo desenvolvido pela Game Science, baseado no romance chinês clássico do século 16, Journey to the West. A jogabilidade foi descrita como semelhante à de Souls.

Data de lançamento inicial: 2023

Desenvolvedor: Game Science Interactive Technology Co.

Distribuidora: Não definido

Gênero: Ação, Aventura, RPG

  • Mínimos:
  • - Sistema Operacional: Windows 7 64-bit
  • - Processor: Intel Core i5-2500K 3.3GHz / AMD FX-8320
  • - Gráficos: AMD Radeon R9 380 or NVIDIA GeForce GTX 960 2GB
  • - VRAM: 2GB
  • - RAM: 8 GB
  • - HDD: 50 GB
  • Recomendado:
  • - Sistema Operacional: Windows 10 64-bit
  • - Processor: Intel Core i7-6700K 4-Core 4.0GHz / AMD Ryzen R5 1600
  • - Graphics: AMD Radeon RX Vega 56 8GB or NVIDIA GeForce GTX 1070
  • - VRAM: 8GB
  • - RAM: 16 GB
  • - HDD: 50 GB

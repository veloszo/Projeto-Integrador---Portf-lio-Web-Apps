Counter-Strike: Global Offensive

Counter-Strike: Global Offensive é um jogo online desenvolvido pela Valve Corporation e pela Hidden Path Entertainment, sendo uma sequência de Counter-Strike: Source. É o quarto título principal da franquia. Data de lançamento inicial: 21 de agosto de 2012

Data de Lançamento: 21 de agosto de 2012

Desenvolvedor: Valve Corporation

Distribuidora: Valve Corporation

Gênero: Ação e FPS

  • Mínimos:
  • - Sistema Operacional: Windows 7 64-bit
  • - Processor: Intel® Core™ 2 Duo E6600 ou AMD Phenom™ X3 8750
  • - Gráficos: 256 MB ou mais de memória
  • - VRAM: 2GB
  • - RAM: 2 GB
  • - HDD: 50 GB
  • Recomendado:
  • - Sistema Operacional: Windows 10 64-bit
  • - Processor: Intel Core i7-6700K 4-Core 4.0GHz / AMD Ryzen R5 1600
  • - Graphics: AMD Radeon RX Vega 56 8GB or NVIDIA GeForce GTX 1070
  • - VRAM: 8GB
  • - RAM: 16 GB
  • - HDD: 50 GB
